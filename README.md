.dotfiles
=========

Requirements
------------

### Brew (for OSX) or APT (for GNU-Linux)

### Git

### Tmux

### Vim

### curl (for installing Plug)

### Plug (for vim plugins)

### Zsh

Installation
------------
```
git clone https://github.com/buduboti/dotfiles.git ~/.dotfiles && ~/.dotfiles/install.sh
```
